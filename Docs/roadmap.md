# Roadmap

## Product direction

Build the most reliable, auditable, and defensible version of OpsPilot's core product:
provenance-first multimodal incident intelligence.

## Near term

- Complete frontend orchestration around backend as single processing source.
- Tighten incident replay UX from persisted job event history.
- Extend artifact trace views (input -> stage -> output lineage).
- Improve confidence calibration transparency in graph UI.

## Mid term

- Add richer policy controls for storage and data retention.
- Add advanced investigation filters over lineage and confidence dimensions.
- Improve operational analytics for incident throughput and failure patterns.

## Long term

- Multi-tenant hardening and enterprise observability packages.
- Automated compliance exports for regulated workflows.

## Non-goals (current stage)

- Premature microservice split.
- Generic platform expansion unrelated to provenance graph value.
- Feature bloat that weakens explainability or chain-of-custody guarantees.
